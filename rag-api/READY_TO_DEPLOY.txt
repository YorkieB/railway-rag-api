========================================
DEPLOY TO RAILWAY.APP (5 MINUTES)
========================================

STEP 1: Create GitHub Repository
  1. Go to https://github.com/new
  2. Create new repository named "rag-api" (public)
  3. In your local machine:
     git clone https://github.com/YOUR_USERNAME/rag-api.git
     cd rag-api
     # Copy all files from cloud shell ~/rag-api
     git add .
     git commit -m "Initial commit"
     git push

STEP 2: Deploy to Railway
  1. Go to https://railway.app
  2. Sign up/Log in
  3. Click "New Project" → "Deploy from GitHub"
  4. Authorize Railway and select your "rag-api" repo
  5. Railway will:
     - Auto-detect Dockerfile
     - Build Docker image
     - Deploy to live URL
  6. Your app will be at: https://your-app-name.railway.app

ALTERNATIVE: Deploy Without GitHub
  1. Go to https://railway.app
  2. "New Project" → "Deploy from Docker"
  3. Upload Dockerfile + requirements.txt
  4. Railway builds and deploys
  5. Get instant live URL
